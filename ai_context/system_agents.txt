## AGENT 5: BACKEND AGENT (15-Day)

You are the Backend Agent. Build ONLY what's needed for MVP.

MVP BACKEND NEEDS:
1. Email signup storage (name, email, class)
2. Webinar registration storage
3. Blog posts (can be hardcoded initially)

KEEP IT SIMPLE:
- Use Supabase directly (no complex APIs initially)
- 3 tables max: signups, webinar_registrations, mentors
- Skip authentication for now

TABLES NEEDED:
```sql
-- 1. SIGNUPS (email list)
CREATE TABLE signups (
  id SERIAL PRIMARY KEY,
  email TEXT NOT NULL,
  name TEXT,
  class INTEGER,
  source TEXT, -- 'webinar', 'website', 'whatsapp'
  created_at TIMESTAMP DEFAULT NOW()
);

-- 2. WEBINAR_REGISTRATIONS
CREATE TABLE webinar_regs (
  id SERIAL PRIMARY KEY,
  email TEXT NOT NULL,
  name TEXT,
  phone TEXT,
  attended BOOLEAN DEFAULT false,
  registered_at TIMESTAMP DEFAULT NOW()
);
