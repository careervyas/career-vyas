# Web Search Strategies — Career Vyas Gap Filling

For each gap type, use these targeted query templates. Replace `{NAME}` with the
entity name (e.g. "NIT Warangal", "WBJEE", "IAS Officer", "B.Pharmacy").
Always prefer the most recent year's data. Note the source URL for every filled value.

---

## CAREER PROFILE Searches

| Gap Field | Query Template | Preferred Sources |
|-----------|---------------|-------------------|
| `salary_insights.entry_lpa` | `{NAME} salary entry level India 2024` | AmbitionBox, Glassdoor, PayScale |
| `salary_insights.mid_lpa` | `{NAME} salary 5 years experience India 2024` | AmbitionBox, LinkedIn Salary |
| `salary_insights.senior_lpa` | `{NAME} salary senior India 2024` | AmbitionBox |

| `day_in_the_life` | `{NAME} typical day routine responsibilities` | Career blogs, Quora, Reddit |
| `eligibility.age_relaxations` | `{EXAM} age limit category relaxation OBC SC ST` | Official UPSC/exam site |
| `entrance_exams` | `how to become {NAME} entrance exam India` | Shiksha, official sites |
| `related_careers` | `careers similar to {NAME} India` | AmbitionBox, Shiksha |
| `faqs` | `{NAME} frequently asked questions India` | Shiksha, Quora, CollegeDekho |

### Special: IAS / UPSC careers
- Pay matrix: `7th pay commission pay matrix IAS officer level 10 to 16`
- Progression: `IAS officer career progression SDM DM Secretary`
- Perks: `IAS officer perks benefits India official`

### Special: Private sector careers (e.g. Software Engineer, CA)
- Salary: Always use AmbitionBox first, then cross-check with Glassdoor
- Growth: `{CAREER} job growth India next 5 years NASSCOM`

### NIRF Data — Most reliable for college stats
Always fetch NIRF data: `https://www.nirf.org/Report/` — contains:
- Teaching-learning & resources
- Research & professional practice
- Graduation outcomes
- Outreach & inclusivity
- Perception score

### JoSAA Cutoffs
For NIT/IIT/IIIT cutoffs, fetch from `josaa.nic.in` or search:
`JoSAA 2024 round 6 {COLLEGE} {BRANCH} closing rank General`

## Source Trust Hierarchy

```
TIER 1 (Official — use directly):
  nirf.org, josaa.nic.in, nta.ac.in, upsc.gov.in, ugc.ac.in,
  aicte-india.org, pci.nic.in, official college websites (.ac.in)

TIER 2 (Reliable aggregators — cross-check with Tier 1):
  shiksha.com, collegedekho.com, careers360.com, collegdunia.com

TIER 3 (Community / salary data — use with note):
  ambitionbox.com, glassdoor.co.in, payscale.com, linkedin.com/salary
  quora.com (for qualitative day-in-life content only)

NEVER USE:
  Random blogs, unverified PDFs, sources without a date
```

## Data Freshness Rules

| Data Type | Max Age Acceptable |
|-----------|-------------------|
| NIRF rankings | 2 years |
| Exam cutoffs | 2 years |
| Placement data | 3 years (note year) |
| Salary data | 2 years |
| Syllabus | 3 years (check for revisions) |
| Important dates | Current year only (mark as "indicative" if prior year) |
| Fee structure | 2 years |

If data is older than the max age, include it but add to `data_quality.gaps_remaining`
with note: "stale_data:{field_name} — year: {year}, needs refresh"
